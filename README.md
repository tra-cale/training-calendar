# 🏃‍♂️ トレーニング管理カレンダー 2025

血圧・体重・トレーニング実績を総合的に管理するWebアプリケーション。日々の健康データを記録し、視覚的に分析できるカレンダー型の管理ツールです。

## 🌐 デモサイト

**🔗 公開URL**: [https://tra-cale.github.io/training-calendar/](https://tra-cale.github.io/training-calendar/)

スマートフォン、タブレット、PCのブラウザから直接アクセスして利用できます。

---

## ✨ 主な機能

### 📊 健康データ管理
- **血圧記録**: 最高血圧・最低血圧の日次記録と推移グラフ
- **体重記録**: 毎日の体重変化を追跡・可視化
- **グラフ表示**: Chart.jsによる美しく分かりやすいデータビジュアライゼーション

### 🏋️ トレーニング記録
5種類のトレーニングに対応し、METs（運動強度）に基づいた正確なカロリー計算を実現:
- **ウォーミングアップ**: 軽いストレッチや準備運動
- **ランニング**: 速度別（4〜12km/h）の消費カロリー自動計算
- **エアロバイク**: 負荷レベル別（軽・中・強）の詳細記録
- **筋トレ**: 腕立て伏せ、腹筋、スクワット、プランク、ランジの5種目対応
- **クールダウン**: ウォーキングによる整理運動

### 📈 統計・分析機能
- **月間サマリー**: 各月の平均値、合計値、達成度を一覧表示
- **年間サマリー**: 1年間のデータを総合的に分析・比較
- **目標設定**: 血圧、体重、トレーニング日数、運動時間、消費カロリーの目標管理
- **達成度表示**: 目標に対する進捗をパーセンテージで可視化
- **トレーニングステータス**: 5段階（記録なし/予定/実行/予定外実行/不実行）で管理

### 🎨 ユーザー体験
- **季節感のあるデザイン**: 月ごとのテーマカラーとアイコン（雪❄️、桜🌸、紅葉🍁など）
- **レスポンシブデザイン**: PC、タブレット、スマートフォンに完全対応
- **直感的なナビゲーション**: アイコンベースのUI、ホバー時にテキスト表示
- **日本の祝日表示**: 2025年の祝日を自動表示
- **プロフィール管理**: 性別、年齢、体重、活動レベルを考慮した個別化計算

## 🚀 使い方

### 💻 オンラインで使用
1. [デモサイト](https://tra-cale.github.io/training-calendar/)にアクセス
2. 初回訪問時にプロフィール情報を入力
3. カレンダー画面で日付をクリックして日々のデータを記録
4. 月間・年間サマリーで進捗を確認

### 📱 スマートフォンでの利用
アプリのように使用できます：
- **iPhone/iPad**: Safariのメニューから「ホーム画面に追加」
- **Android**: Chromeのメニューから「ホーム画面に追加」

### 🖥 ローカルで動かす
```bash
# リポジトリをクローン
git clone https://github.com/tra-cale/training-calendar.git

# ディレクトリに移動
cd training-calendar

# index.htmlをブラウザで開く（または直接home.htmlを開く）
```

## ⚠️ 既知の制限事項

### データ保存
- **LocalStorageの容量制限**: 約5MB（通常の使用では十分）
- **ブラウザ依存**: ブラウザやデバイスごとに別々のデータが保存される
- **データ消失リスク**: ブラウザのキャッシュクリアや再インストールでデータが失われる可能性

### 互換性
- **ブラウザ要件**: モダンブラウザ（Chrome、Safari、Firefox、Edge）の最新版を推奨
- **JavaScript必須**: JavaScriptが無効だと動作しません

### 推奨事項
- 定期的にスクリーンショットを撮影してデータをバックアップ
- 年間サマリーを印刷または PDF 保存
- 今後実装予定のエクスポート機能でデータを定期的に保存
